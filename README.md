# Simple Web Tracker

This is a very easy project for beginners.

You can use this repository to:
- Track daily tasks
- Write small notes
- Practice editing files
- Build a habit of daily GitHub commits

How to use:
1. Open any file
2. Add a small update
3. Save the file
4. Commit your change

Goal:
Make small progress every day.

Notes:
Write your daily tracking notes below.
